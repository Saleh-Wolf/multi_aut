# Laravel Multi Authentication System

A simple Laravel project that implements **Multi Authentication (Admin & User)** using Laravel Breeze.

## Features

- User Registration & Login
- Admin & User Dashboard
- Middleware for Admin Access
- Image Upload during Registration
- Form Validation using Form Requests
- Authentication using Laravel Breeze

## Technologies Used

- Laravel 12
- PHP
- MySQL
- Laravel Breeze
- Blade Templates
